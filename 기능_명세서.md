# "현지인처럼" 앱 상세 기능 명세서 (PRD) v1.0

이 문서는 '현지인처럼' 앱의 주요 기능에 대한 상세 명세를 정의합니다.

---

## 1. 현지인의 추천 (Local's Pick)

### 1.1. 추천 등록/수정 (Create/Edit)
사용자가 새로운 장소를 추천하거나 기존 추천을 수정하는 기능입니다.

*   **필수 입력 필드:**
    *   `장소 선택`: 지도에서 검색하거나 핀을 직접 찍어 위치 지정. (주소 텍스트 자동 변환)
    *   `사진`: 1장 이상, 최대 5장까지 업로드. (첫 번째 사진이 대표 이미지)
    *   `장소 이름`: 텍스트, 최대 30자.
    *   `카테고리`: 음식점, 카페, 자연, 쇼핑, 문화, 산책로 등 predefined 리스트에서 최소 1개, 최대 3개 선택.
    *   `한 줄 요약`: 텍스트, 최대 50자. (리스트 및 요약 카드에 노출)
*   **선택 입력 필드:**
    *   `추천 스토리`: 텍스트, 최대 1,000자. (Markdown 형식 지원)
    *   `나만의 팁`: 텍스트, 최대 200자.
*   **시스템 자동 입력:**
    *   `작성자 프로필`: 작성 시점의 사용자 프로필 정보.
    *   `작성/수정 일시`: 서버 시간 기준.
*   **액션 버튼:**
    *   `발행하기`: 모든 필수 필드 충족 시 활성화. 등록 플로우 완료.
    *   `임시저장`: 작성 중인 내용을 임시 저장. (서버에 저장되며, '내 정보 > 임시저장 글'에서 확인 가능)
    *   `취소`: 작성 중인 내용 삭제 후 이전 화면으로 이동.

### 1.2. 추천 조회 (View)
사용자가 등록된 추천 콘텐츠를 보는 기능입니다.

*   **지도 위 요약 뷰 (Map Card View):**
    *   지도 위의 핀을 탭하면 화면 하단에 작은 카드가 노출됨.
    *   구성 요소: 대표 `사진`, `장소 이름`, `한 줄 요약`, `카테고리`.
*   **상세 뷰 (Detail View):**
    *   요약 카드를 탭하면 전체 화면으로 전환됨.
    *   구성 요소: 모든 필드 정보(사진 슬라이더, 장소 이름, 카테고리, 스토리, 팁 등), 작성자 프로필 정보(사진, 닉네임, 인증 뱃지), '유용해요' 개수.
    *   하단 액션 바: `나만의 지도에 저장`, `길찾기`(외부 지도 앱 연동), `공유하기`.
    *   상단 액션: `'유용해요' 누르기`(❤️ 아이콘), `신고하기`.
    *   본인 게시물일 경우, `수정` 및 `삭제` 옵션 노출.

## 2. 사용자 프로필
사용자의 정보를 표시하고 관리하는 페이지입니다.

*   **구성 요소:**
    *   `프로필 사진` 및 `닉네임` (수정 가능)
    *   `거주/활동 지역` 및 `현지인 인증 뱃지` (시스템 표시)
    *   `관심사 태그` (선택 및 수정 가능)
    *   Tab 메뉴:
        *   `내 추천`: 내가 작성한 '현지인 추천' 게시물 목록.
        *   `내 지도`: 내가 생성한 '나만의 지도' 목록.
        *   `저장한 추천`: '유용해요'를 누른 다른 사람의 추천 목록.
*   **설정 메뉴:**
    *   `알림 설정`, `로그아웃`, `회원 탈퇴`.

## 3. 동네 Q&A 게시판 (MVP)
특정 지역에 대해 질문하고 답변하는 게시판 기능입니다.

*   **질문(Question) 등록:**
    *   필드: `지역 태그`(필수), `제목`, `내용`, `사진`(최대 3장, 선택).
*   **답변(Answer) 등록:**
    *   필드: `내용`.
*   **주요 기능:**
    *   `답변 채택`: 질문자는 가장 도움이 된 답변 1개를 '채택'할 수 있음. 채택된 답변은 최상단에 고정 노출.
    *   `공감하기`: 질문과 답변에 '공감' 버튼(👍)으로 유용성을 표현.
    *   `정렬`: 최신순, 공감순으로 목록 정렬.

---

## 4. 나만의 지도
관심 있는 '현지인 추천'들을 모아 나만의 여행 지도를 만드는 기능입니다.

*   **기능:**
    *   `지도 생성`: 새로운 지도 생성 및 이름 부여.
    *   `추천 저장`: '현지인 추천' 상세 페이지에서 '나만의 지도에 저장' 버튼을 통해 기존 지도에 추가.
    *   `지도 조회`: 저장된 장소들이 핀으로 찍힌 지도 뷰와 목록 뷰 제공.
    *   `지도 공유`: 생성된 지도를 다른 사람에게 공유할 수 있는 URL 생성. (초기에는 읽기 전용으로 공유)
