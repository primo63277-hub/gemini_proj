# "현지인처럼" 앱 사용자 흐름 및 와이어프레임 v1.0

이 문서는 '현지인처럼' 앱의 핵심 사용자 흐름(User Flow) 2가지와 각 화면의 구조를 텍스트 기반 와이어프레임으로 정의합니다.

---

## Flow 1: 여행자가 현지 추천을 발견하고 저장하는 흐름

**목표:** 여행자가 앱을 열어 주변의 추천 장소를 발견하고, 마음에 드는 장소를 자신의 여행 지도에 추가한다.

1.  **[화면: 홈 (지도)]**
    *   앱 실행 시 나타나는 첫 화면.
    *   사용자의 현재 위치를 중심으로 지도가 보이며, 주변의 '현지인 추천' 장소들이 핀(📍)으로 표시됨.
    *   하단에는 `[홈]`, `[검색]`, `[Q&A]`, `[내 정보]` 탭 바 존재.
    *   화면 우측 하단에는 `[내 위치로]` 버튼과 `[필터]` 버튼이 있음.
    *   **사용자 액션:** 지도 위의 핀(📍) 중 하나를 탭한다.

2.  **[화면: 추천 요약 카드]**
    *   화면 하단에서 해당 장소의 정보가 담긴 작은 카드가 나타남.
    *   카드에는 `대표 사진`, `장소 이름`, `한 줄 요약`, `카테고리`가 표시됨.
    *   **사용자 액션:** 요약 카드를 탭한다.

3.  **[화면: 추천 상세]**
    *   선택한 추천 장소의 전체 정보가 표시되는 화면.
    *   **사용자 액션:** 내용을 확인 후, 하단의 `'나만의 지도에 저장'(⭐️) 아이콘`을 탭한다.

4.  **[화면: 지도 선택 팝업]**
    *   기존에 만들어 둔 '나만의 지도' 목록이 나타남. (예: "서울 빵집 투어", "제주 서쪽 여행")
    *   목록 최상단에는 `[+ 새 지도 만들기]` 옵션이 있음.
    *   **사용자 액션:** 저장하고 싶은 지도(예: "서울 빵집 투어")를 선택한다.

5.  **[완료]**
    *   "나의 지도에 저장되었습니다" 라는 확인 메시지(Toast)가 잠시 나타났다가 사라짐.
    *   사용자는 계속해서 상세 화면을 보거나 뒤로 가기를 할 수 있음.

## Flow 2: 현지인이 새로운 장소를 추천하는 흐름

**목표:** 현지인이 다른 사람들에게 소개하고 싶은 새로운 장소를 앱에 등록한다.

1.  **[화면: 홈 (지도)]**
    *   화면 중앙 하단(탭 바 위)에 있는 `[+ 추천하기]` 플로팅 액션 버튼(FAB)을 탭한다.

2.  **[화면: 추천 등록 (1/3) - 장소 선택]**
    *   `'어떤 장소를 추천하시나요?'` 라는 안내 문구와 함께 지도 검색 화면이 나타남.
    *   상단에는 `[장소 이름 또는 주소 검색]` 입력창이 있음.
    *   사용자는 검색을 하거나, 지도를 직접 움직여 핀으로 위치를 지정할 수 있음.
    *   **사용자 액션:** 위치 지정을 완료하고 하단의 `[이 장소로 추천하기]` 버튼을 탭한다.

3.  **[화면: 추천 등록 (2/3) - 정보 입력]**
    *   `사진`, `카테고리`, `한 줄 요약`, `추천 스토리`, `나만의 팁` 등을 입력하는 폼(Form) 화면.
    *   필수 항목과 선택 항목이 구분되어 있음.
    *   **사용자 액션:** 필수 항목을 모두 입력하고 하단의 `[미리보기]` 버튼을 탭한다.

4.  **[화면: 추천 등록 (3/3) - 미리보기]**
    *   실제 앱에 게시될 모습과 똑같이 디자인된 미리보기 화면이 나타남.
    *   **사용자 액션:** 내용을 최종 확인하고 하단의 `[발행하기]` 버튼을 탭한다.

5.  **[완료]**
    *   `"🎉 새로운 추천이 성공적으로 등록되었습니다!"` 와 같은 축하 메시지와 함께 완료 화면이 나타남.
    *   `[내가 쓴 글 확인하기]` 버튼과 `[홈으로 가기]` 버튼이 제공됨.
